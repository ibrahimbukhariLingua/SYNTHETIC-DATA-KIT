Le code Afep-Medef devra évoluer vers un vrai code de Place


Le code Afep-Medef achoppe une nouvelle fois sur sa légitimité. Dans sa synthèse des réponses à la consultation publique sur la révision du code Afep-Medef, le professeur Bertrand Fages rappelle que « certains investisseurs ou agences de conseil en vote […] semblent ne pas s’accommoder » du choix du législateur de confier l’élaboration du code à des «organisations représentatives des entreprises», selon les termes de l’article L. 255-37-4, 8° du Code de commerce, et reconnaît que «cette critique est récurrente». De fait, «il est regrettable que cette consultation entérine un code de gouvernance réalisé par les seuls émetteurs, confie Michael Herskovich, responsable de la gouvernance d’entreprise chez BNP Paribas AM. Les investisseurs étrangers ne comprennent pas ce choix français et ont d’emblée une vision très négative de notre code, occultant ainsi ses avancées. Une modification législative, qui pourrait être portée par la loi Pacte, est nécessaire. » BNPP AM propose d’ouvrir la rédaction du code à d’autres acteurs que les seules organisations représentatives des entreprises. Le code Afep-Medef est «une exception culturelle française, explique Jean-Philippe Desmartin, directeur de l’investissement responsable chez Edmond de Rothschild AM. Beaucoup d’entreprises se sont mises en conformité avec ce code, mais il ne répond pas forcément aux attentes des investisseurs. D’ailleurs, les étrangers sont surpris que ce code soit le référentiel de Place. Les entreprises sont la propriété des actionnaires, et donc des investisseurs. Pourtant ce sont les dirigeants qui fixent les règles en l’espèce ». Aussi, puisque le patronat promet une révision du code tous les deux ans, « nous militons pour l’émergence d’un véritable code de Place, en mettant par exemple autour de la table Afep-Medef et AFG , poursuit Jean-Philippe Desmartin. Cela nous donne le temps pour prendre en compte l’ensemble des attentes des investisseurs avant de débuter la décennie 2020 avec un code de gouvernance plus conforme aux standards internationaux ». «Je déplore que le code soit écrit par les seuls émetteurs , confie Caroline Ruellan , présidente de Sonj, cabinet de conseil en gouvernance et stratégie. De plus, il aurait été judicieux d’attendre le vote de la loi Pacte avant de modifier le code Afep-Medef. » Les règles élémentaires pas toujours appliquées Ceci dit, la Place salue les révisions régulières du code en fonction des demandes du marché, des investisseurs, voire du gouvernement. « Si le code évolue dans la bonne direction, il en vient à émettre des évidences. Signe que les règles élémentaires – assiduité des administrateurs, abstention de vote et de présence au débat dans une situation de conflit d’intérêts, etc. – sont encore loin d’être appliquées par tous , ajoute Caroline Ruellan. Néanmoins, cette révision envoie un message très fort à la Place : les conseils ne sont plus des chambres d’enregistrement. Cette plus grande implication des administrateurs nécessite aussi de relever les jetons de présence. » Parmi les nouveautés 2018, la prise en compte par le conseil de la création de valeur sur le long terme, en tenant compte des dimensions sociales, sociétales et environnementales des activités de l’entreprise. Sur le fond, «nous sommes satisfaits de la prise en compte par le conseil des enjeux RSE, dans la foulée du rapport Notat-Senard. Cela va dans le sens de l’histoire», poursuit Michael Herskovich. Mais si le code intègre davantage la RSE, «il peine à appréhender l’extra-financier , estime Caroline Ruellan. Or, tout risque extra-financier sera un risque financier demain ». Le dialogue actionnarial à affiner Au cœur des réflexions du Haut comité de gouvernement d’entreprise ( HCGE ) l’an dernier, le dialogue actionnarial entre dans le code. « Nous saluons l’instauration formelle d’un dialogue avec les actionnaires , ajoute Michael Herskovich. Toutefois, le code aurait pu aller plus loin en ne limitant pas ce rôle au président ou à l’administrateur référent. Dans les faits, nous parlons souvent avec les présidents de comité ». Le dialogue entre le conseil et les investisseurs « pose encore beaucoup de problèmes de mise en œuvre , tempère Caroline Ruellan. Si le code révisé donne un signal positif aux investisseurs, il appartient aux entreprises de construire un canal de communication avec un président non exécutif ou un administrateur référent qui maîtrise l’exercice, et sans rompre l’égalité entre les actionnaires ». Du côté des rémunérations, « l’encadrement des clauses de non-concurrence est également positive. Dans notre politique, nous demandons que les sociétés soient beaucoup plus claires et explicites sur les circonstances d’attribution des rémunérations exceptionnelles », précise Michael Herskovich. En revanche, « nous souhaitions que le code recommande la séparation des pouvoirs, entre président et directeur général, mais l’Afep-Medef se refuse à prendre position , ajoute Michael Herskovich. Ce qui aurait conduit les sociétés optant pour une structure non dissociée à s’expliquer. Or, la règle du ‘comply or explain’ devient ‘comply and explain’ avec des recommandations du code peu ambitieuses ». Tant l’Afep-Medef que le HCGE peuvent faire évoluer le code de gouvernance sur nombre de questions. Notamment, « le vrai sujet n’est pas l’indépendance des administrateurs mais l’absence de conflits d’intérêts. Une question encore peu traitée pour les exécutifs qui cumulent plusieurs mandats d’administrateurs hors groupe », précise Caroline Ruellan. Enfin, le patronat a fait un effort en élargissant le HCGE, qui passera de sept à neuf membres en fin d’année. Mais «les représentants des émetteurs seront encore majoritaires et les investisseurs auront au mieux deux sièges», regrette Michael Herskovich.
