GFM lance une plateforme de données sur les fonds


GFM vient de lancer Globalfunddata, une plateforme gratuite de données quantitatives sur plus de 34.000 fonds (hedge funds, fonds long only, ETF…). Sur www.globalfunddata.com, on peut trouver des informations sur la stratégie, le gérant, le code ISIN, la date de lancement, les frais, etc.
