«On ne pourra plus jouer sur l’ambiguïté sur le marché des changes»


L’Agefi : Quel est l’intérêt du tout nouveau code global pour le marché des changes publié le 25 mai? Imène Rahmouni-Rousseau : Ces dernières années, il y a eu de nombreux scandales sur le marché des changes. Il y avait besoin de réaffirmer les règles de bonne conduite pour garantir que les prix ne sont pas manipulés. Deuxièmement, le marché des changes étant global, il était très important pour les banques centrales que l’effort s’applique sur toutes les grandes places financières. Enfin, le code va s’appliquer à tous les participants du marché, pas uniquement aux grandes banques. Le code qui était en vigueur en Europe était celui de l’association ACI et s’appliquait à l’ensemble des marchés. Le code global est plus court et spécifique. Que pouvez-vous faire pour généraliser l’adoption du code? Les banques centrales exigeront de leurs contreparties qu’elles respectent le code d’ici à un an. Si ce n’est pas le cas, la banque centrale se réservera le droit de ne plus échanger avec cette contrepartie. Cela va concerner toutes les grandes banques systémiques de la planète. Il existe aussi des comités régionaux sur le marché des changes, qui sont chapeautés par les banques centrales et qui ne sont pas seulement composés des banques mais aussi de gérants d’actifs et d’entreprises non financières. Ils pourront exiger que leurs membres respectent le code. Enfin, la Banque de France a été en contact avec les associations de Place, telles que l’association bancaire ACI-AFTB et l’association française de la gestion financière (AFG). Elles devraient promouvoir l’adoption du code auprès de leurs membres. Adopter le code ne risque-t-il pas d’être compliqué pour les petits acteurs? C’était l’une des préoccupations des gérants d’actifs et des entreprises non financières mais le code inclut un principe de proportionnalité. Autrement dit, les recommandations s’appliquent à tous mais les modalités d’application varient en fonction du type et du degré d’activité de chaque établissement sur le marché des changes. Les banques centrales estiment qu’un délai d’un an pour appliquer le code est raisonnable sachant que certains chapitres avaient déjà été publiés. Pour les acteurs, il y aura des ajustements à faire mais ce ne sera pas une révolution. Quelles bonnes pratiques seront compliquées à adopter? Il y a un champ qui mérite d’être étudié avec attention, c’est celui des échanges d’informations. Une banque ne peut jamais communiquer avec un client sur une transaction qu’elle a conduite avec d’autres clients de manière précise. Par contre, il est possible de communiquer une information anonymisée et agrégée. L’autre chapitre clé du code concerne l’exécution des ordres. Le code recommande que les banques soient davantage transparentes avec leurs clients sur la tarification des ordres. Un des problèmes que faisaient remonter les petits acteurs est que certaines banques adoptaient des tarifications décalées par rapport aux standards de marché. Un autre sujet important concerne la pratique du dernier regard (last look) qui, par le passé, a été utilisée à l’encontre des intérêts des clients. Sur les marchés électroniques, une banque a une période de quelques secondes pour confirmer ou rejeter un ordre pour lequel le client s’est déjà engagé. Le code précise que cette période doit être utilisée uniquement à des fins de gestion des risques de la part de la banque ou dans l’intérêt du client. Comment être sûr que les pratiques changeront s’il n’y a pas de tiers pour faire respecter le code? Il s’agit d’un code volontaire mais nous encourageons les acteurs à faire une déclaration d’engagement à respecter le code, à la publier et à l’utiliser dans leurs relations avec leurs contreparties. Il y a une réflexion en cours sur la création de registres publics qui rassembleraient ces déclarations. Certaines autorités de marché et certains superviseurs bancaires considèrent qu’il serait utile de pouvoir se référer à ce code. De façon générale, la marche à franchir pour dévier des pratiques acceptables sera beaucoup plus haute dans la mesure où les règles ont été acceptées par tous. On ne pourra plus jouer sur l’ambiguïté. Comment le code peut-il restaurer la confiance dans le marché? Le code n’est pas une baguette magique qui va arrêter les amendes mais on peut imaginer qu’il aura un impact réel sur le marché. Le Global Foreign Exchange Committee réfléchit à la mise en place d’une enquête à l’automne pour mesurer l’impact du code sur la confiance dans le marché, qui serait renouvelée un an plus tard. Si l’impact n’est pas au rendez-vous, il y aura toujours la possibilité de renforcer le code.
